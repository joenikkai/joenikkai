## 🛠️ Active Projects

### [pjdev](https://github.com/joenikkai/pjdev)
A high-performance TUI-based IDE/Editor built with C++ and Ncurses.
- **Triple Mode Editing:** INSERT, REPLACE, and EXECUTE (Command) modes.
- **Syntax Highlighting:** Support for Rust, C++, Python, JS, PHP, Bash, and ASM.
- **Smart Navigation:** Intelligent visual line wrapping and word processor key support.
- **Integrated Shell:** Execute bash commands directly from the editor or directory mode.
- **Journaling:** Robust crash protection with auto-saving and journal management.
